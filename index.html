Map Maker

Supported file format: .txt

File structure:

  • FIRST LINE: WIDTH HEIGHT
  • SUBSEQUENT LINES: Tile values, in the following format: (L,N)
    • where L is V, F, M, S, R and N is between 1-5 OR
    • L and N are both 0.

WIDTH and HEIGHT are both integers. You should have HEIGHT number of rows, with WIDTH number of (L,N) pairs separated by a single space.

Users may choose to utilize the C++ program found within this GitHub repo to generate a map file.

EXAMPLE FILE:

5 4
(F,1) (M,1) (0,0) (R,1) (0,0)
(0,0) (M,1) (0,0) (R,1) (0,0)
(M,1) (M,1) (M,1) (R,1) (R,1)
(M,1) (R,1) (0,0) (C,1) (V,1)
