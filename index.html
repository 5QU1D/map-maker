Map Maker

Supported file format: .txt

File structure:

  • FIRST LINE: WIDTH HEIGHT
  • SUBSEQUENT LINES: Tile values, in the following format: (L,N)
    • where L is V, F, M, S, R and N is between 1-5 OR
    • L and N are both 0.

WIDTH and HEIGHT are both integers. You should have HEIGHT number of rows, with WIDTH number of (L,N) pairs separated by a single space.

V stands for Valley, F stands for Forest, M stands for Mountain, S stands for Settlement, and R stands for River.

Users may choose to utilize the C++ program found within this GitHub repo to generate a map file.

PLEASE NOTE: The largest advisable size within current program limitations is somewhere in the range of 77 by 77 tiles. Larger than that is likely to cause the generator to appear to be infinitely loading and require a hard page refresh before the generator can be used for any other files. This issue will be addressed in later patches some time after 5/16/2025.

EXAMPLE FILE:

5 4
(F,4) (M,3) (M,1) (R,5) (V,3)
(M,3) (M,5) (M,3) (R,4) (V,2)
(M,4) (M,5) (S,5) (R,3) (V,1)
(M,2) (M,4) (M,3) (R,2) (0,0)
